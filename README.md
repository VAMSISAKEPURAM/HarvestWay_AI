# 🌾 HarvestWay AI — Intelligence For Every Acre

> **HarvestWay AI** is a premium, state-of-the-art agricultural intelligence ecosystem designed to put world-class agronomical insights, predictive AI models, and local market connection directly into the hands of farmers. 

Built with Next.js (App Router), React 19, TypeScript, Tailwind CSS v4, and Shadcn/UI, the platform delivers a fast, fluid, and fully responsive multi-lingual dashboard designed for "one-thumb use" right in the field.

## 🚀 Key Features

HarvestWay AI is engineered with **eight AI-native modules** crafted specifically for the daily decisions of farmers:

### 1. 🔍 Crop Disease Detection (Sanjeevani AI)
* **What it does**: Scan crop leaves via an upload and instantly identify crop diseases with precise confidence levels.
* **Impact**: Provides instant pesticide/treatment recommendations with exact dosage windows (e.g. *Mancozeb 0.25% • 7-day spray window*).
* **Integration**: Fully integrated with the [Sanjeevani AI service](https://sanjeevani-ai-frontend.vercel.app/).

### 2. 🌱 Best Crop Recommendation
* **What it does**: Directs farmers to the most suitable and profitable crops to grow based on real-time soil, meteorological, and economic parameters.
* **Impact**: Ranks crops with percentage compatibility (e.g., *Maize: 94% fit*, *Soybean: 88% fit*).
* **Integration**: Powered by the [Best Crop Recommendation engine](https://best-crop-recomender.vercel.app/).

### 3. 📈 Crop Yield Prediction
* **What it does**: Forecasts prospective yield output per hectare (e.g., *4.6 t/ha*) by combining historical farming data with meteorological patterns.
* **Impact**: Enables better resource allocation and crop planning.

### 4. ☀️ Weather-Based Advisory
* **What it does**: Hyper-local weather monitoring that translates forecasts into actionable farm work items.
* **Impact**: Recommends smart adjustments to your schedule to protect investments (e.g., *"Sunny • Light rain at 18:00 — Delay pesticide spray by 6 hours"*).

### 5. 🤖 24/7 Multilingual AI Assistant
* **What it does**: A friendly, smart conversational chatbot that provides instant answers to complex farming queries.
* **Impact**: Real-time water conservation, soil advice, and cultivation tips (e.g., *"Soil moisture is 28%. Irrigate tomorrow at 6:00 AM for 25 minutes — saves 18% water"*).

### 6. 📊 Mandi Price Prediction
* **What it does**: Provides predictive market intelligence for regional Mandi (market) crop prices.
* **Impact**: Anticipates price movement trajectories (e.g., *₹2,480/qtl, expected +6.4% in the next 14 days*) to tell farmers exactly when to sell.

### 7. 🤝 Farmer Support Network ("Hire a Farmer")
* **What it does**: Bridges the gap between cities and rural communities, enabling urban consumers to sponsor natural farming and source direct farm-fresh products.

### 8. 🏪 Direct Farmer-to-Buyer Marketplace
* **What it does**: Enables direct-to-buyer transactions, bypassing middlemen.
* **Impact**: Fairer compensation for farmers and faster payment turnarounds.

## 💻 Local Development Setup

To run HarvestWay AI locally, follow these simple steps:

### 1. Prerequisites
Make sure you have Node.js (v18.x or above) installed on your system.

### 2. Clone and Install
```bash
# Clone the repository
git clone <your-repository-url>
cd HarvestWay-AI

# Install package dependencies
npm install
```

### 3. Run Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

### 4. Build & Production Start
```bash
# Build the production bundle
npm run build

# Start the Next.js production server
npm run start
```

### 5. Code Quality (Linter & Formatting)
```bash
# Run ESLint to check for code issues
npm run lint

# Run Prettier to automatically format code
npm run format
```
